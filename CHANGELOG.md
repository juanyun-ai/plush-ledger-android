# 更新日志

## 0.8.0 - 2026-06-14

- 账单页新增右上角搜索，修正收入/支出筛选展示，月汇总和每日分组同时展示收入与支出。
- 统计页按参考图重做暖色汇总、支出构成、周支出趋势和分类排行。
- 我的页、设置页、用户信息页、预算管理、分类管理和关于我们继续按新增参考图重构。
- 数据导出改为先弹窗确认并选择保存位置。
- 通知提醒固定展示 5 条消息，版本更新消息保留确认下载入口。
- 分类管理三条杠支持轻点下移、双击上移、长按确认删除，并把分类顺序保存到本地和云端同步队列。
- 底部导航选中项改为橙色圆角背景和白色文字图标。

## 0.6.5 - 2026-06-12

- 顶部月份控件改为可点击日历，标题说明保持单行并优化尺寸。
- 首页、账单、记账、统计和“我的”进一步按参考图统一布局与暖橙毛绒视觉。
- 选中按钮统一使用橙色背景和白色文字，底部导航同步适配。
- 修复头像上传遇到过期 JWT 时返回 403 的问题，自动刷新登录会话后重试。
- 新增 App 内页面返回栈，系统侧滑返回会优先回到上一个页面。
- 新增发布版本自动写入官方信箱消息的数据库触发器。
- 新增 Android Studio Compose 预览示例，支持在电脑查看关键页面。
- 公开仓库前移除本地云端配置和个人信息，加入禁止商用许可。

## 0.6.0 - 2026-06-12

- 新增启动自动检查版本和设置页手动检查更新。
- 新增 APK 下载、SHA-256 校验和系统安装器更新流程。
- 新增相册头像选择、压缩上传、私有云存储和跨设备头像恢复。
- 信箱从底部导航迁入“我的”，保留官方消息和建议反馈。
- 新增独立账单页，底部导航调整为首页、账单、记账、统计、我的。
- 按 3D 毛绒参考图重新设计暖白背景、橙色主操作、彩色分类图标和信息面板。
- 新增 Supabase `app_versions` 表、`avatars` 私有桶和 `app-releases` 发布桶。

## 0.5.0 - 2026-06-12

- 完成邮箱验证码注册、设置密码及邮箱密码登录。
- 增加用户名密码本地模式，本地数据不上传云端。
- 完成 Room 本地优先存储和 Supabase 后台同步。
- 增加首页、记账、统计、信箱和“我的”页面。
- 增加日期选择、环形图、柱状图、搜索、预算、分类及账户管理。
- 增加头像昵称、管理员铭牌、永久会员入口和用户建议反馈。
- 增加可选 PIN、生物识别、防截图、CSV 导出、退出及账号注销。
- 部署账号注销 Edge Function，并收紧数据库 RLS 与触发函数权限。
- 微信/QQ 登录及微信/支付宝支付等待开放平台资质，当前不产生真实登录或扣款。
